# JSCAD + JustinSDK + NopSCAD Assembly

This repository demonstrates a mixed-library scene where JSCAD composes geometry from both linked libraries through `@justinsdk/...` and `@nopscad/...` imports.

Suggested linked libraries:

- `JSCAD Modeling`
- `justinsdk`
- `NopSCAD`
